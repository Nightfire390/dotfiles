### Linux AwesomeWM dotfiles

Current arch setup dotfiles for workstations, generally suited for bigger screen, requires minor adjustments for smaller screens. 

```
git clone https://github.com/Nightfire/dotfiles
cd dotfiles

./script.sh
```

##### Contains:
- Zsh
- NeoVim
- Alacritty
- AwesomeWM
- Rofi
